# koa-restful-quick-start

koa-restful 快速启动工程

## 快速开始

```sh
# 克隆项目
git clone https://github.com/WangMingHua111/koa-restful-quick-start.git
# 进行工作目录
cd koa-restful-quick-start
# 安装依赖
npm install
# dev模式启动
npm run dev
```

## 本地开发

```sh
npm run dev
```

## 构建生产环境

```sh
npm run build
```

## 运行服务

```sh
npm run serve
```
